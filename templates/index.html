🚀 DevOps Lab - Flask with Swagger

📋 Project Overview

Complete REST API with Swagger documentation. All endpoints are fully documented and testable through Swagger UI.

📖 View Swagger Documentation

🔌 Available Endpoints

GET /api/users - List all users
GET /api/users/{id} - Get user by ID
POST /api/users - Create new user
PUT /api/users/{id} - Update user
DELETE /api/users/{id} - Delete user
GET /api/users/filter/by-age?min_age=25 - Filter by age
GET /api/users/stats/summary - Get statistics

🧪 Quick Test

Try these commands:

curl http://localhost:5000/api/users

curl -X POST http://localhost:5000/api/users -H "Content-Type: application/json" -d '{"name":"Alice","email":"alice@email.com","age":28}'
